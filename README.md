# Adquisición de Datos 📊

Este directorio contiene los Jupyter Notebooks, scripts y recursos correspondientes a la asignatura de **Adquisición de Datos**. El objetivo principal de este espacio es documentar el código, los procesos de extracción, transformación y carga (ETL), y el análisis de datos desarrollado durante las prácticas del semestre.

## 🛠️ Tecnologías y Requisitos
Para ejecutar los notebooks de este repositorio sin problemas, se recomienda tener instalado lo siguiente:
* **Python 3.x**
* **Jupyter Notebook** o **JupyterLab**

* Librerías principales de Python:
  * `pandas`
  * `numpy`
  * `requests`
  * `BeautifulSoup` / `Selenium` (para web scraping)
  * `matplotlib` / `seaborn` (para exploración visual)

Puedes instalar las dependencias generales ejecutando:
```bash
pip install pandas numpy jupyter requests beautifulsoup4 matplotlib seaborn
