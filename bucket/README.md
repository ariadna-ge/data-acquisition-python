# Adquisición de Datos: Cloud Storage (S3/GCS)
*Conectividad y gestión de datos en almacenamiento de objetos en la nube.*
Implementación de conectores para la ingesta de archivos almacenados en buckets de la nube (Amazon S3 / Google Cloud Storage).

## Características:
- Configuración de credenciales seguras mediante variables de entorno.
- Operaciones de lectura y escritura remota (Cloud Storage / S3).
- Sincronización de carpetas locales con buckets remotos.
- Manejo de archivos en formatos binarios y comprimidos.
