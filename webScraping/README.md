# Adquisición de Datos: Web Scraping
*Extracción de datos desde el DOM (Document Object Model) de sitios web.*
Proceso de extracción de datos no estructurados desde páginas web mediante peticiones HTTP y parseo de contenido HTML. Incluye manejo de estructuras JSON extraídas.

## Características:
- Uso de librerías para parseo de HTML (BeautifulSoup/lxml).
- Identificación de selectores CSS y Xpath para extracción precisa.
- Implementación de delays (time.sleep) para evitar bloqueos por parte del servidor.
- Limpieza de texto y normalización de datos extraídos.
